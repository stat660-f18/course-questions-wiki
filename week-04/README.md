[Course Textbook Chapter 8, Problem 3]
- Question (anguyen152-stat660): What is an example of SAS valid new format's name ? 
- Answer (anguyen152-stat660): see $GENDERLABEL in this example 
PROC FORMAT;
    VALUE $GENDERLABEL
	  "M"   = "Male"
	  "F"   = "Female";
RUN;
